Task management system

Task management system

Зробити веб-додаток - своєрідний список справ. Тут можна створювати завдання на день, додавати до них різну інформацію, як опис, очікуваний час виконання, підзавдання і т.д. Також кожному завданню можна встановити статус: заплановано, виконується, виконано та ін. Також кожне завдання може мати підзавдання, які визначають ступінь виконання основного. Ще до завдань можна залишати коментарі, змінювати їх, видаляти.

Чому навчитеся:
  • робота з веб-фреймворком у вашій мові
  • робота з БД
  • робота з UI (js, html, css)
Як покращити:
  • зробити, щоб завдання можна було прикріплювати файли
  • додати авторизацію користувача
  • можливість мати окремий робочий простір для кожного користувача
  • можливість додавати користувачів до свого робочого простору з різними правами
  • можливість візуалізувати прогрес із завдань (timeline, графік тощо) для всієї команди у робочому просторі
Chat

Chat

Веб-програма типу Telegram, яку користувач може запустити, і приєднатися до інших користувачів, щоб поспілкуватися з ними, надсилаючи їм текстові повідомлення.

Чому навчитеся:
  • робота з WebSocket
  • робота з веб-фреймворком з вашої мови
  • робота з UI (html, css, js)
  • робота з БД (я б віддав перевагу тут NoSQL)
Як покращити:
  • додати канали, куди можуть підключатися одразу групи користувачів
  • додати надсилання файлів
  • додати реакцію повідомлення
  • додати форматування тексту повідомлень
  • додати нотифікації про повідомлення
Clone Instagram

Clone Instagram

Веб-додаток, куди користувач може викладати свої фотографії, а інші користувачі – його передплатники – можуть побачити ці фотографії у своїй стрічці, оцінити їх, залишити коментар.

Чому навчитеся:
  • робота з веб-фреймворком у вашій мові
  • робота з БД
  • робота з UI (js, html, css)
  • робота з файлами
  • робота із запланованими завданнями (scheduled jobs) для генерації стрічки
Як покращити:
  • можливість збирати статистику про перегляди та лайки за весь час (графік, загальні цифри) для користувача. Корисно для знаменитостей
  • кеш для популярних користувачів, щоб швидко отримувати їх фото
  • cтрічка подій (хто на вас підписався, хто поставив лайк під вашим фото або залишив коментар)
  • альбоми (можливість публікувати кілька фото в одному пості та гортати їх)
